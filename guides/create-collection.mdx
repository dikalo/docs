---
title: "Guide : Créer une collecte"
description: "Guide pas à pas pour créer ta première collecte de paiement."
---

Ce guide t'accompagne dans la création de ta première collecte, du paramétrage jusqu'au premier paiement reçu.

## Étape 1 : Connecte-toi au dashboard

Va sur [taramoney.com](https://www.taramoney.com) et connecte-toi. Si tu n'as pas encore de compte, crée-en un gratuitement.

## Étape 2 : Crée une nouvelle collecte

Dans le menu latéral, clique sur **Collectes**, puis sur **Nouvelle collecte**.

## Étape 3 : Remplis les informations

- **Nom** : choisis un nom clair. Ex : "Cotisation mensuelle Janvier"
- **Description** : explique brièvement à quoi sert la collecte
- **Montant** : fixe (ex: 5000 FCFA) ou laisse vide pour un montant libre
- **Date de fin** : optionnelle, mais recommandée

## Étape 4 : Ajoute des options

### Cadeaux

Si tu actives les cadeaux, les participants pourront offrir des articles prédéfinis. Parfait pour les anniversaires et mariages.

### QR Code

Active le QR code pour que les gens puissent payer en scannant.

## Étape 5 : Partage

Une fois créée, ta collecte a un lien unique. Copie-le et partage-le sur :

- **WhatsApp** : envoie le lien à tes contacts ou dans un groupe
- **Facebook** : publie le lien sur ta page
- **Email** : inclus le lien dans ta newsletter
- **Flyer** : imprime le QR code et affiche-le

## Étape 6 : Suis les paiements

Dans le dashboard, tu vois les paiements arriver en temps réel. Chaque paiement affiche le nom du payeur, le montant, et l'heure.
