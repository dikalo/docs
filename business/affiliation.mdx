---
title: "Programme d'affiliation"
description: "Gagne de l'argent en recommandant Tara à d'autres commerçants."
---

Le programme d'affiliation Tara te récompense pour chaque nouveau business que tu amènes sur la plateforme.

<Tip>
  Chaque affilié reçoit une commission récurrente sur les transactions de ses filleuls. Plus ils encaissent, plus tu gagnes.
</Tip>

## Comment ça marche ?

### Pour les affiliés

<Steps>
  <Step title="Demande ton code affilié">
    Dans le dashboard, va dans **Affiliation** et active ton compte affilié. Tu reçois un code unique.
  </Step>
  <Step title="Partage ton code">
    Partage ton code ou ton lien d'affiliation avec d'autres commerçants.
  </Step>
  <Step title="Gagne des commissions">
    Tu touches un pourcentage sur chaque transaction des commerçants que tu as parrainés.
  </Step>
</Steps>

## Dashboard d'affiliation

Le dashboard affilié affiche :

- **Code affilié** unique
- **Nombre de filleuls** actifs
- **Commissions** gagnées (total et par mois)
- **Top filleuls** par volume de transactions
- **Lien de parrainage** à partager

## Soumettre une affiliation

Les nouveaux affiliés doivent soumettre une demande. Dans le dashboard :

1. Va dans **Affiliation → Soumettre**
2. Remplis le formulaire avec tes informations
3. L'équipe Tara valide ta demande

## Pour les admins

L'interface admin d'affiliation permet de :

- Valider ou rejeter les demandes d'affiliation
- Configurer les taux de commission
- Suivre les performances par affilié
- Générer des rapports

## Paiement des commissions

Les commissions sont versées automatiquement sur ton compte Tara chaque mois.
