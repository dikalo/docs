---

L'endpoint Paiements te permet de lister et rechercher les transactions liées à ton business.

## Endpoint

```
GET https://www.dklo.co/api/tara/payments
```

## Headers

| Header | Valeur |
|--------|--------|
| `Authorization` | `Bearer ta_clé_privée` |

## Paramètres de requête

| Paramètre | Type | Obligatoire | Description |
|-----------|------|-------------|-------------|
| `businessId` | string | Oui | Ton identifiant business |
| `status` | string | Non | Filtrer par statut (`SUCCESS`, `PENDING`, `FAILED`) |
| `startDate` | string | Non | Date de début (ISO 8601) |
| `endDate` | string | Non | Date de fin (ISO 8601) |
| `collectionId` | string | Non | Filtrer par collecte |
| `limit` | number | Non | Nombre de résultats (défaut: 50) |
| `offset` | number | Non | Pagination |

## Exemple de requête

```bash
curl -X GET "https://www.dklo.co/api/tara/payments?businessId=ton_business_id&status=SUCCESS&limit=20" \
  -H "Authorization: Bearer ta_clé_privée"
```

## Exemple de réponse

```json
{
  "total": 150,
  "limit": 20,

  "payments": [

```
