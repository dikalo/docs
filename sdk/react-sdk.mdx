Le SDK React officiel de Tara fournit des composants prêts à l'emploi et un client HTTP pour intégrer les paiements Mobile Money.

<Tip>
  **npm** : `@taramoney/react-sdk` v0.1.0 — Compatible React ≥17 et Next.js (App Router). Zero dépendance externe.
</Tip>

## Installation

```bash
npm install @taramoney/react-sdk
# ou
yarn add @taramoney/react-sdk
```

## Configuration

### Provider

Enveloppe ton application avec `<TaraProvider>` :

```tsx
import { TaraProvider } from "@taramoney/react-sdk";

function App() {
  return (
    <TaraProvider
      config={{
        apiKey: "ta_clé_publique",
        businessId: "ton_business_id",
        environment: "production",
      }}
    >
      {/* Ton app */}
    </TaraProvider>
  );
}
```

### TaraConfig

| Paramètre | Type | Obligatoire | Description |
|-----------|------|-------------|-------------|
| `apiKey` | `string` | Oui | Ta clé publique Tara |
| `businessId` | `string` | Oui | Ton identifiant business |
| `environment` | `"sandbox" \| "production"` | Non | Environnement (défaut: `"sandbox"`) |
| `baseUrl` | `string` | Non | URL de base de l'API (override) |
| `qrBaseUrl` | `string` | Non | URL de base pour les QR (défaut: `https://taramoney.com/pay/`) |
| `useProxy` | `boolean` | Non | Utiliser le proxy `/api/tara-proxy` pour éviter les CORS |

### Proxy Next.js (recommandé)

Pour éviter les blocages CORS navigateur, utilise le proxy intégré :

**1. Crée la route proxy :**

```ts
// app/api/tara-proxy/route.ts
export { GET, POST, PUT, DELETE } from "@taramoney/react-sdk/server";
```

**2. Active le mode proxy :**

```tsx
<TaraProvider
  config={{
    apiKey: "ta_clé_publique",
    businessId: "ton_business_id",
    environment: "production",
    useProxy: true,
  }}
>
  <App />
</TaraProvider>
```

### Proxy standalone (Vite / CRA / React pur)

```bash
npx @taramoney/react-sdk proxy
# → http://localhost:3001

# Puis configure :
# baseUrl: "http://localhost:3001/api/tara"
# useProxy: false
```

## Composants

### QrCode

QR code de paiement scannable. Encode un lien `https://taramoney.com/pay/{collectionId}?p={amount}`.

```tsx
import { QrCode } from "@taramoney/react-sdk";

// Avec montant fixe
<QrCode collectionId="06706" amount={4000} size={256} />

// Sans montant (libre)
<QrCode collectionId="06706" size={200} />
```

#### Props

| Prop | Type | Obligatoire | Description |
|------|------|-------------|-------------|
| `collectionId` | `string` | **Oui** | Identifiant de la collecte |
| `amount` | `number` | Non | Montant en FCFA (libre si absent) |
| `size` | `number` | Non | Taille en pixels (défaut: `256`) |
| `qrBaseUrl` | `string` | Non | URL de base du lien (défaut: `https://taramoney.com/pay/`) |
| `instructionText` | `string` | Non | Texte sous le QR (défaut: `"Scannez ce code pour effectuer le paiement."`) |
| `className` | `string` | Non | Classe CSS additionnelle |

### EmbedPay

Intègre la page de paiement complète en iframe. **Pas besoin de `TaraProvider`** — le composant est autonome.

```tsx
import { EmbedPay } from "@taramoney/react-sdk";

// Thème Tara par défaut
<EmbedPay
  collectionId="06706"
  amount={4000}
  accentColor="ff4400"
  backgroundColor="faf9f7"
  textColor="1a1a1a"
  height={600}
  onSuccess={(data) => console.log("✅ Payé !", data)}
  onError={(err) => console.error("❌", err)}
/>

// Thème sombre personnalisé
<EmbedPay
  collectionId="06706"
  amount={4000}
  accentColor="00ff88"
  backgroundColor="1a1a1a"
  textColor="ffffff"
  height={600}
/>
```

#### Props

| Prop | Type | Obligatoire | Description |
|------|------|-------------|-------------|
| `collectionId` | `string` | **Oui** | Identifiant de la collecte |
| `amount` | `number` | Non | Montant en FCFA |
| `accentColor` | `string` | Non | Couleur d'accentuation hex sans # (défaut: `ff4400`) |
| `backgroundColor` | `string` | Non | Couleur de fond hex sans # (défaut: `faf9f7`) |
| `textColor` | `string` | Non | Couleur du texte hex sans # (défaut: `1a1a1a`) |
| `payBaseUrl` | `string` | Non | URL base page de paiement (défaut: `https://taramoney.com/pay/`) |
| `height` | `number \| string` | Non | Hauteur de l'iframe (défaut: `600`) |
| `width` | `number \| string` | Non | Largeur de l'iframe (défaut: `100%`) |
| `onSuccess` | `(data) => void` | Non | Callback après paiement réussi (postMessage) |
| `onError` | `(error) => void` | Non | Callback après erreur (postMessage) |
| `className` | `string` | Non | Classe CSS additionnelle |

L'URL générée par l'iframe :
```
https://taramoney.com/pay/06706?p=4000&accent=00ff88&bg=1a1a1a&text=ffffff&isFrame=true
```

## Hook `useTara()`

Accède au contexte Tara depuis n'importe quel composant enfant de `<TaraProvider>`.

```tsx
import { useTara } from "@taramoney/react-sdk";

function MesPaiements() {
  const { config, client, isReady } = useTara();

  const loadPayments = async () => {
    const result = await client.fetchPayments({
      businessId: config.businessId,
      limit: 5,
    });
    console.log(result);
    // { total: 150, limit: 5, offset: 0, payments: [...] }
  };

  return <button onClick={loadPayments}>Charger</button>;
}
```

### Retour du hook

| Champ | Type | Description |
|-------|------|-------------|
| `config` | `TaraConfig` | Configuration active |
| `client` | `TaraMoneyClient` | Instance du client HTTP |
| `isReady` | `boolean` | Provider monté et prêt |

## Client API (`TaraMoneyClient`)

Le client expose toutes les méthodes pour interagir avec l'API Tara.

### `client.mobilePay(params)`

Déclenche un paiement Mobile Money.

```tsx
const response = await client.mobilePay({
  apiKey: config.apiKey,
  businessId: config.businessId,
  productId: "prod_001",
  productName: "Collecte scolaire",
  productPrice: 100,
  phoneNumber: "2376xxxxxxxx",
  // network — optionnel, laisser vide pour auto-détection
  // "wave" pour Wave (Sénégal, CIV, Burkina)
  network: "",
  webHookUrl: "https://tonsite.com/webhook",
});
// { message: "API_ORDER_SUCESSFULL", status: "SUCCESS", vendor: "ORANGE_CAMEROON" }
// Wave : { ..., authUrl: "https://..." } — redirige l'utilisateur vers cette URL
```

| Paramètre | Type | Obligatoire | Description |
|-----------|------|-------------|-------------|
| `apiKey` | `string` | **Oui** | Ta clé publique |
| `businessId` | `string` | **Oui** | Ton identifiant business |
| `productId` | `string` | **Oui** | Identifiant du produit/collecte |
| `productName` | `string` | **Oui** | Nom du produit/collecte |
| `productPrice` | `number` | **Oui** | Montant en devise locale |
| `phoneNumber` | `string` | **Oui** | Numéro du payeur (format `2376xxxxxxxx`) |
| `network` | `string` | Non | Code réseau. Vide = auto-détection, `"wave"` pour Wave |
| `webHookUrl` | `string` | Non | URL appelée par le webhook après confirmation |

### `client.fetchPayments(params)`

Récupère la liste des paiements.

```tsx
const result = await client.fetchPayments({
  businessId: config.businessId,
  status: "SUCCESS",
  limit: 20,
  offset: 0,
});
// { total: 150, limit: 20, offset: 0, payments: Transaction[] }
```

| Paramètre | Type | Obligatoire | Description |
|-----------|------|-------------|-------------|
| `businessId` | `string` | **Oui** | Ton identifiant business |
| `status` | `TransactionStatus` | Non | Filtrer par statut |
| `startDate` | `string` | Non | Date début (ISO) |
| `endDate` | `string` | Non | Date fin (ISO) |
| `collectionId` | `string` | Non | Filtrer par collecte |
| `limit` | `number` | Non | Résultats par page |
| `offset` | `number` | Non | Pagination |

### `client.getTransactionStatus(reference)`

Vérifie le statut d'une transaction.

```tsx
const tx = await client.getTransactionStatus("txn_abc123");
// { reference: "txn_abc123", amount: 5000, status: "SUCCESS", ... }
```

### `client.createPaymentLink(params)`

Crée un lien de paiement.

```tsx
const link = await client.createPaymentLink({
  apiKey: config.apiKey,
  businessId: config.businessId,
  amount: 5000,
  description: "Frais de dossier",
  reference: "DOSSIER-001",
  redirectUrl: "https://tonsite.com/confirmation",
});
// { status: "SUCCESS", paymentLink: "https://pay.taramoney.com/link/abc123", reference: "DOSSIER-001" }
```

### `client.getQrCodeUrl(collectionId, amount?, qrBaseUrl?)`

Génère l'URL du lien de paiement pour QR code (local, pas d'appel API).

```tsx
const url = client.getQrCodeUrl("06706", 4000);
// https://taramoney.com/pay/06706?p=4000
```

## Types TypeScript

### Transaction

```ts
interface Transaction {
  reference: string;          // Référence unique
  amount: number;             // Montant en FCFA
  status: "SUCCESS" | "PENDING" | "FAILED";
  phoneNumber: string;        // Format 2376xxxxxxxx
  productId?: string;         // Identifiant du produit
  createdAt: string;          // Date ISO
}
```

### MobileNetwork

```ts
// Le type network accepte n'importe quel code réseau (string).
// Laisser vide pour l'auto-détection, "wave" pour Wave.

type MobileNetwork = string;
```

### Réseaux courants

```ts
import { MOBILE_NETWORKS } from "@taramoney/react-sdk";
// {
//   ORANGE_CAMEROON: "ORANGE_CAMEROON",

```

### MobilePayResponse

```ts
interface MobilePayResponse {
  message: string;
  status: TransactionStatus;
  vendor: string;
  authUrl?: string;  // Présent uniquement pour les paiements Wave
}
```
