---
title: "QR Codes"
description: "Génère des QR codes de paiement pour tes collectes. Scannables via WhatsApp, Telegram ou navigateur."
---

Chaque collecte Tara peut générer un QR code qui encode un lien de paiement direct. Le client scanne et paie en quelques secondes.

<Tip>
  Le QR code encode un lien `https://taramoney.com/pay/{collectionId}?p={amount}`. Une fois scanné, le client arrive sur la page de paiement Tara et choisit son moyen de paiement (Orange Money, MTN, Wave, etc.).
</Tip>

## Via le SDK React

```tsx
import { QrCode } from "@taramoney/react-sdk";

// QR code avec montant fixe — 256px
<QrCode collectionId="06706" amount={4000} size={256} />

// QR code à montant libre
<QrCode collectionId="06706" size={200} />
```

### Props

| Prop | Type | Défaut | Description |
|------|------|--------|-------------|
| `collectionId` | `string` | — | Identifiant de la collecte (**obligatoire**) |
| `amount` | `number` | — | Montant en FCFA (libre si absent) |
| `size` | `number` | `256` | Taille en pixels |
| `qrBaseUrl` | `string` | `https://taramoney.com/pay/` | URL de base du lien de paiement |
| `instructionText` | `string` | `"Scannez ce code pour effectuer le paiement."` | Texte sous le QR |
| `className` | `string` | — | Classe CSS additionnelle |

## Via le client SDK

```tsx
import { useTara } from "@taramoney/react-sdk";

function MonQrCode() {
  const { client } = useTara();
  const paymentUrl = client.getQrCodeUrl("06706", 4000);
  // https://taramoney.com/pay/06706?p=4000

  // Utilise le lien pour générer un QR code via n'importe quel service
  const qrImageUrl = `https://api.qrserver.com/v1/create-qr-code/?size=256x256&data=${encodeURIComponent(paymentUrl)}`;

  return <img src={qrImageUrl} alt="QR Code Tara" />;
}
```

## Lien de paiement

Le QR code encode un lien direct vers la page de paiement Tara :

```
https://taramoney.com/pay/{collectionId}?p={amount}
```

| Paramètre | Description |
|-----------|-------------|
| `collectionId` | Identifiant de la collecte |
| `p` | Montant en FCFA (optionnel) |

Exemple : `https://taramoney.com/pay/06706?p=4000`

## Personnalisation (EmbedPay)

Pour une expérience intégrée, utilise le composant `EmbedPay` qui affiche la page de paiement complète en iframe avec couleurs personnalisables :

```tsx
import { EmbedPay } from "@taramoney/react-sdk";

<EmbedPay
  collectionId="06706"
  amount={4000}
  accentColor="ff4400"
  backgroundColor="ffffff"
  textColor="1a1a1a"
  height={600}
/>
```
